# **simple-zhihu-spider**
============================	
## **[Usage](#Usage)**
1. Create a SuperSession object
	- session = SuperSession()
2. Use method do to update your session cookies
	- session.do()
3. Use get(or other requests.Session methods) to get response
	- session.get(url, headers..)
